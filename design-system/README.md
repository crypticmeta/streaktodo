# Design System Reference

This directory contains the visual reference material for Streak Todo.

## How to use it

The files here are HTML/CSS/JS prototypes, not production code. Use them as a
visual reference when implementing or refining the real app. Match the design
direction, tokens, spacing, and interaction patterns, but keep production
behavior in the app codebase.

When working from these files:

- read the primary files in `design-system/project/`
- follow imported CSS and shared component files so token usage is clear
- treat the prototypes as reference material, not as code to copy directly

## Bundle contents

- `design-system/README.md` — this file
- `design-system/project/` — HTML prototypes, assets, and reference components
