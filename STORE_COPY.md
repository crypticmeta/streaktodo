# Streak Todo Store Copy

Draft Play Store copy for the first public Android release.

## App title

Streak Todo

## Short description

Plan tasks, keep streaks, and stay on schedule.

## Full description

Streak Todo is a focused planner for people who want a clear daily task list
without clutter.

Create tasks quickly, organize them into categories, set reminders, and repeat
important routines. Track what is due today, what is coming up, and what needs
attention next.

Core features:

- Fast task capture from a floating action button
- Categories for work, personal tasks, wishlist items, and more
- Subtasks for breaking larger tasks into smaller steps
- Due dates, times, reminders, and repeat rules
- Calendar view for seeing tasks by day
- Streak and completion tracking
- Light and dark theme support
- Local-first task storage on your device

Streak Todo is built to stay simple:

- no attachment system
- no upgrade paywalls
- no forced collaboration

Use it as a personal scheduler first. Add structure to your day, keep important
habits visible, and finish more without turning your planner into a second job.
